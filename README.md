### 安装
 
>1. 安装vim (你得先有vim) <br>
>2. 安装git <br>
>3. clone配置文件： git clone https://github.com/ymc023/vimIDE.git <br>
>4. cd vimIDE && bash vimIDEsetup.sh <br>
>5. 等待脚本自动安装插件完成! <br>
